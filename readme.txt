== Description ==

Careme is a premium, high-performance WordPress theme designed specifically for modern medical clinics, dental practices, and healthcare providers. It features a clean, professional aesthetic with integrated appointment booking, doctor profiles, and a robust health library.

== Features ==

* Fully Responsive & Mobile Optimized
* Premium Medical Design with High-Quality AI Assets
* Live Search Functionality & AJAX Appointment Forms
* WordPress Customizer Integration for Real-time Branding
* High-Conversion Hero Sections & Count-up Stats
* Before/After Transformation Slider for Dental/Surgical results
* Comprehensive Health Library (Blog) infrastructure
* Accessible & Semantic HTML5/CSS3 (Sass)

== Installation ==

1. In your WordPress admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme.

== Frequently Asked Questions ==

= How do I change the clinic phone number? =
Go to Appearance > Customize > Contact Information and update the Phone field.

= How do I manage the services grid? =
Initial version uses template parts. Future updates will include Full Site Editing (FSE) support.
